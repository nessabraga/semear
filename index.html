Projeto SEMEAR

Semeando hoje para colher amanhã!

Nosso projeto temo objetivo apresentar uma opção viável para resolver os prblemas do auto custo na comercialização de frutas, leguminosas e hortaliças a partir do recolhimento do lixo orgânico doméstico, encaminhando-o para a produção de adubo, que será reutilizado para auxiliar no plantio e fertilização, refletindo no menor preço de mercado para você!
